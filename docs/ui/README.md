## デザイン原則

### 1. AIファースト
- エージェントの活動状況が一目で分かる
- MCP接続状態の可視化
- セッション履歴へのアクセス

### 2. シンプル & 機能的
- Linearライクなミニマルデザイン
- キーボードショートカット重視
- 高速なナビゲーション

### 3. macOSネイティブ
- SwiftUI標準コンポーネント活用
- サイドバー + メインコンテンツ構成
- ダークモード対応

## キーボードショートカット

| ショートカット | アクション |
|---------------|-----------|
| `⌘ N` | 新規タスク作成 |
| `⌘ ⇧ N` | 新規プロジェクト作成 |
| `⌘ ⇧ A` | 新規エージェント作成 |
| `⌘ 1` | プロジェクト一覧へ |
| `⌘ 2` | タスクボードへ |
| `⌘ 3` | エージェント管理へ |
| `⌘ ,` | 設定を開く |
| `⌘ F` | 検索 |
| `↑ / ↓` | タスク選択 |
| `Enter` | タスク詳細を開く |
| `Space` | タスクステータス変更 |

---

## レスポンシブ対応

| ウィンドウ幅 | レイアウト |
|-------------|-----------|
| < 600px | サイドバー非表示、ハンバーガーメニュー |
| 600-900px | サイドバー縮小（アイコンのみ） |
| > 900px | フルサイドバー表示 |
